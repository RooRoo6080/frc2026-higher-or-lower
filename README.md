# [Higher or Lower for FRC](https://frc2026-higher-or-lower.onrender.com)
Randomly selects robots from the 2026 game Rebuilt with pictures from [TBA](https://www.thebluealliance.com/) and compares EPA from [Statbotics](https://www.statbotics.io/).

https://frc2026-higher-or-lower.onrender.com

## Run Locally

1. Install dependencies:
   `npm install`
2. Set the `TBA_API_KEY` in `.env` to your TBA API key
3. Run the app:
   `npm run dev`
